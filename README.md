# RaspberryPipico_DPS310_LCDdisplay

## 方針

- 飛行機や登山時に環境データを取得したい。
- データをLCDで表示させ、またSDカードに保存したい。
- CO₂濃度をモニタリング

## 具体的な内容

- 測定データ、Max,Min、グラフを表示。重ねてグラフに表示
  - DPS310：温度、気圧
  - 温湿度センサ：温度、湿度
  - CO₂センサ：CO₂濃度
- 気圧、温度はレンジ変えず。気圧は変える。
- （ビープサウンダー：リポバッテリーの過充電過放電対策）
- TP4056モジュール
- SDカードに各種データ保存。

- 手のひらサイズにコンパクトに収納。
- リポバッテリーを収納
  - 飛行機内で充電不可になる可能性があるため
  - TypeCで充電できるように。
  - 奈良に行くときに大阪日本橋でリポ購入？容量は小さくてOKなはず。
  - 電圧をディスプレイ上に表示。3.3V入力を分岐させてIOピンに接続させればOK
  - リポ→VSYSピン(2.3~5.5V)

### 電源関係
- リポ→VSYSピン(2.3~5.5V)
- リポ電源が3.2V以下で画面に警告を出す。3.0V以下でビープ音。
- 同様に過充電も。
- TP4056モジュールの購入がよさそう。Amazon等で安く売ってる。
―――――――――――――――――――――
VSYSピンは2.3V〜5.5Vを受け入れる設計になっているため、リポバッテリーの電圧をそのまま入力できます。

Pico（無印）の場合: GPIO29が内部的にVSYSの電圧を測定するために予約されています。外部回路（分圧抵抗）を組まずに、プログラムからVSYSの電圧をチェックできます。
